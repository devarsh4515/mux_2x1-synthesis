# 2×1 Multiplexer using Verilog HDL

## Overview
This repository contains the RTL implementation of a 2×1 Multiplexer using Verilog HDL. The project also includes synthesis using Yosys and the synthesized circuit visualization.

## Repository Contents
- RTL Verilog source code
- Testbench
- Yosys synthesis script
- Synthesized netlist
- Yosys DOT graph
- Synthesized circuit image

## Features
- RTL design in Verilog HDL
- Functional verification using testbench
- Logic synthesis using Yosys
- Circuit visualization

## Tools Used
- Verilog HDL
- Yosys
- Git & GitHub
